에러 : Because 프로젝트명 requires SDK version >=3.0.5 <4.0.0, version solving failed.
Jul 10, 2023

1. 문제

Flutter 프로젝트를 끌어와서 VSCode > pubspec.yml 파일에서 Get Packages 하려고 하니 디버그 콘솔에 이러한 문구가 뜬다.

Resolving dependencies...
The current Dart SDK version is 3.0.3.

Because introduce_e1if requires SDK version >=3.0.5 <4.0.0, version solving failed.
Exited (1)

2. 원인

내 Dart 버전이 3.0.3 인데 3.0.5 ~ 4.0.0 버전을 요구하고 있어서 그렇다

..
environment:
  sdk: '>=3.0.5 <4.0.0'
..

3. 해결방법

Dart SDK를 올려주면 해결된다. VSCode Terminal에서 flutter upgrade 해주자
